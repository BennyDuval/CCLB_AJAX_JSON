How To

Use the dropdown button in order to select one of the five properties. You can change your selection at any time. The site will then display an image of the property, the properties location on Google Maps, and the associated XML data in table form.

A challenge with this project was the amount of data to display, and that certain data points had no or limited information. But putting it in table form makes it much more digestible.
